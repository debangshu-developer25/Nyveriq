# 🚀 Nyveriq

A modern **Task & Project Management System** built with **ASP.NET Core Web API**, **React**, **TypeScript**, and **SQL Server**.

---

# ✨ Features

- 🔐 JWT Authentication
- 👤 User Management
- 📁 Project Management
- ✅ Task Management
- 📋 Kanban Board
- 📊 Dashboard
- 💬 Task Comments
- 📎 File Attachments
- 📜 Audit Logs
- 🎨 Modern Login & Signup UI
- 🔒 Role-Based Authorization

---

# 🛠 Tech Stack

## Backend

- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication

## Frontend

- React
- TypeScript
- Axios
- CSS

## Database

- Microsoft SQL Server

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/debangshu-developer25/Nyveriq.git
```

## Backend

```bash
cd TaskFlow.API

dotnet restore

dotnet ef database update

dotnet run
```

## Frontend

```bash
cd taskflow-ui

npm install

npm start
```
